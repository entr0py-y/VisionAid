# Smart Wearable Assistive Device for Visually Impaired People

## College Major Project - 2026

### Description
A comprehensive website showcasing a smart wearable assistive device that uses AI and computer vision to help visually impaired individuals navigate their environment independently.

### Tech Stack
- **Backend**: Node.js, Express.js
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **No external frameworks or libraries**

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the server:
```bash
npm start
```

or

```bash
node server.js
```

3. Open your browser and navigate to:
```
http://localhost:3000
```
